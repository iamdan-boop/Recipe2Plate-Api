# Requirements

* [MYSQL](https://www.mysql.com/)
* [JAVA 15 or JDK 15](https://www.oracle.com/java/technologies/javase/jdk15-archive-downloads.html)



## Installation

Install all the dependencies of Recipe2Plate Project

```bash
./mvnw install
```
